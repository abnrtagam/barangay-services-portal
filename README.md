# Barangay Bulua: Digital Service Portal

![Barangay Bulua Logo](frontend/public/logo.png)

A premium, full-stack digital governance solution designed for **Barangay Bulua**. This system streamlines resident interactions, automates document requests, and provides a secure platform for complaint management and appointment booking.

> [!NOTE]
> **Academic Project:** This system was developed as part of a school project to demonstrate modern development practices. Beyond the academic requirements, this project served as a deep learning journey—building this system from the ground up allowed for a hands-on exploration of full-stack architecture, premium UI/UX design, and the impact of digital governance on local communities.

## 🚀 Key Features

### 💻 Resident Web Portal (React)
- **Modern 50/50 Auth Layout:** High-end, responsive registration and login experience.
- **Digital Profile:** Manage residency data and track verified status.
- **Complaint System:** Submit and track grievances with automated status updates.
- **Appointment Booking:** Securely book slots for barangay services.

### 📱 Mobile Application (Flutter)
- **On-the-go Access:** Full parity with web features for mobile users.
- **Push Notifications:** Real-time updates on appointment and complaint statuses.
- **Secure Authentication:** Integrated with the same high-security backend.

### 🛡️ Admin Management (Laravel)
- **Centralized Dashboard:** Manage residents, verify accounts, and process requests.
- **Document Generation:** Automated processing of barangay certificates.
- **Analytics & Reports:** Insightful data on community needs and service performance.

## 📂 Project Structure

- `/frontend`: React.js web portal (Vite).
- `/flutter_app`: Cross-platform mobile application.
- `/backend`: Node.js/Express API service.
- `/laravel-backend`: Core administrative and document processing engine.
- `/database`: SQL scripts and schema updates.
- `/docs`: API collections, progress logs, and system documentation.

## 🛠️ Tech Stack

- **Frontend:** React, Vite, Vanilla CSS (Premium Custom Design)
- **Mobile:** Flutter, Provider for State Management
- **Backend:** Node.js, Express, Laravel (PHP)
- **Database:** MySQL
- **Security:** JWT Authentication, Bcrypt Hashing, OTP Verification
